> **Release date:** 2026-03-06

> **Branch:** `main`

---

## Что это

Полноценная B2B платформа для онбординга инженеров через AI-powered knowledge graph. Компания загружает кодовую базу и документацию, AI строит knowledge graph, генерирует ежедневные 15-минутные миссии, и адаптивно обучает инженеров через tri-agent coaching (Strategist → Designer → Coach).

Эволюция от v0.1.0-mvp (5 сервисов, 113 тестов) до полноценной платформы (10 сервисов, 1343 теста).

---

## Что работает (end-to-end)

### Flow 1: Инженер проходит онбординг
```
Регистрация → Выбор организации → Dashboard (7 блоков) →
→ Получение daily mission → Сессия с AI Coach (5 фаз) →
→ Повышение mastery → Flashcard review → Trust level up
```

### Flow 2: Tech Lead управляет KB
```
Создание организации → Загрузка документов / GitHub repos →
→ AI извлекает концепты → Knowledge graph строится →
→ Team analytics (heatmap coverage, bottleneck report) →
→ Настройка LLM провайдера → Billing (Stripe)
```

### Flow 3: Поиск знаний
```
Smart Search (Cmd+K) → Query Router →
→ Internal: RAG semantic search по KB организации
→ External: Gemini search по публичным источникам
→ Concept Hub: Obsidian-like страница концепта
```

### Flow 4: Преподаватель создаёт контент
```
Seller app → Создание курса → Модули + уроки →
→ AI генерация: outline, lessons, quizzes →
→ Promotions, bundles, coupons → Analytics
```

### Flow 5: Геймификация
```
Ежедневная активность → Streak → XP →
→ Leaderboard (per course) → Badges →
→ Pretest → Adaptive difficulty → Certificate
```

## Честная оценка

**Что получилось хорошо:**
- Полный B2B flow: от загрузки кодовой базы до AI coaching с mastery tracking
- 1343 теста дают надёжную safety net
- Clean Architecture во всех сервисах — навигируемый и расширяемый код
- Tri-agent AI coaching — персонализированные миссии из реальной KB
- Knowledge graph с concept mastery — measurable learning outcomes
- Dark Knowledge UI — функциональный и стильный интерфейс
- Monitoring с первого дня (Prometheus + Grafana)
- Rust для performance-critical путей (gateway, search, chunking)

**Что нужно проверить под нагрузкой:**
- RAG semantic search latency на больших KB
- AI Coach response time (зависит от Gemini API)
- Concurrent mission generation throughput
- pgvector index performance на 100K+ chunks
- API gateway proxy overhead

**Что не реализовано:**
- Real-time messaging (WebSocket) — polling only
- OAuth / Social login
- Video transcoding — URL-based storage only
- MCP Server для внешних AI агентов
- Multi-region deployment
- ClickHouse аналитика
